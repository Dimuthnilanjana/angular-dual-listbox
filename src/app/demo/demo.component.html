Angular Dual List Box

A flexible, customizable dual list box component for Angular applications

How to use

Select items from the left list and click the arrow buttons to move them to the right list. You can also use the search feature to filter items in either list.

Tip: Hold Ctrl (or Cmd on Mac) while clicking to select multiple items.
